## Installation
1. Clone the repository to your local computer using the following command:

   ```shell
   https://github.com/mhaidar10/japan-travel.git
   ``` 
2. Install the Library with the following command:

    ```shell
    npm install
    
    ```

## Usage
1. Navigate to the Project Directory:

    ```shell
    cd japan-travel
    ```
    
2. Run Project:
   ```shell
    npm run dev
    ```
